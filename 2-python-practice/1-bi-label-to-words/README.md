将BI标注序列转换为词序列
========================

任务描述
--------

给__字序列__标注B、I标记是中文分词的常用方法。
B表示这个字是一个词的开始，比如`热爱`的`热`；
I表示这个字是一个词的中间，比如`热爱`的`爱`。

在指导字序列的BI标注序列后，我们就可以将这个序列转化为词序列。
举例来讲，对于句子`我是中国人`，假设它的BI标注序列为`BBBII`，那么句子的分词结果就是`我/是/中国人`。

现在的任务是在给定字序列的BI标注序列的条件下，输出对应的词序列。
